# Kompecell
## App Description: 
Kompecell se yon app mobil ki la pou oryente itilizate, pou yo kapab jwenn bon aparey elektronik nan meye pri, fe chwa de kompecell ou pap regret,benefisye yon rabe 10% sou chak pwodwi ou ht nan Kompecell.

### Must Have 
- Itilizatè a dwe kapab kreye yon kont oswa konekte fasil.

- App lan dwe gen yon katalòg pwodwi elektwonik (telefòn, tablèt, òdinatè, elatriye).

- Dwe gen yon sistèm rechèch ak filtraj pa kategori, pri, oswa mak.

- Yon sistèm peye sekirize (kat, transfè, elatriye).

- Yon notifikasyon konfimasyon acha apre chak tranzaksyon.

### Maybe Have 
- Yon asistan vityèl ki ede itilizatè a chwazi bon aparèy selon bezwen li.

- Posibilite pou sove pwodwi nan “wishlist”.

- Itilizatè a dwe kapab konpare pwodwi yo pou konnen kiyès ki pi bon an.

### Would be Nice to Have 

- Rabe 10% sou chak pwodwi achte atravè Kompecell.

- Yon pwogram fidèlite kote itilizatè yo touche pwen pou chak acha.

- Livrezon gratis pou yon kantite minimòm acha.
